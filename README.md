# AAE
## サイト概要
家電製品を紹介する体験会イベント告知サイトです。
Twitterのアカウントを通じて気軽にイベントを作り、自由に参加できるサイトです。
製品に実際に触った意見や情報を発信し、紹介することができます。

### サイトテーマ

製品PRを兼ねた体験会イベント告知
### テーマを選んだ理由
家電量販店はイベントを開く機会があり、告知をする手段を広げることができれば面白いと考えたのがきっかけです。イベント運営の手段として告知アプリの作成を作ろうと考えました。

### ターゲットユーザ

イベント告知
### 主な利用シーン
イベント運営を宣伝し、参加を呼び掛けたいとき。
自分が開きたいイベントを作成し、Twitterアカウントを通じて告知宣伝をする時。
