### テーマを選んだ理由
<...>
人と関われる橋渡しになるアプリを作ってみたいと思ったからです。コロナ禍によって人と関わる機会が少なくなった他、出会いの場も作りようがない状況だと思います。
人が集まれるような機会を作ることに繋がる手段としてイベント告知を考えました。
### ターゲットユーザ
<...>
自分の好きなことで催しものを開き人と関わってみたい方。
### 主な利用シーン
<...>
自分が開きたいイベントを作成し、SNSを通じて告知宣伝をする時。
